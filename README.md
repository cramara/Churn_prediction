# Churn Prediction

## Installation

### Create a virtal environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Sur Windows :**
```bash
venv\Scripts\activate
```

**Sur Linux/Mac :**
```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the app

```bash
streamlit run app_streamlit.py
```
