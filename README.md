# Churn Prediction

## Installation

### Créer un environnement virtuel

```bash
python -m venv venv
```

### Activer l'environnement virtuel

**Sur Windows :**
```bash
venv\Scripts\activate
```

**Sur Linux/Mac :**
```bash
source venv/bin/activate
```

### Installer les dépendances

```bash
pip install -r requirements.txt
```
